How it works!

Take a link

You have a long link that you want to post on social media or send it via email, but you're running out of characters or when someone tries to click on the link in email it is not working because it wraps around and the email client renders it useless.

Shrink it!

Go to {{ request.META.HTTP_HOST }} and you can submit your link via the form. You will be given a new short link to use anywhere you want. You can also create multiple links to the same location and then you can use these different links to track how visitors find your site.

Use the API

send a http get request to {{ request.META.HTTP_HOST }}/api/create/ with a url parameter and you'll receive a json request back with your new short link. You can automate your short link creation against our API.
